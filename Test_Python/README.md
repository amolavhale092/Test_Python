# Rock, Paper, Scissors Game

This project is a simple implementation of the classic game Rock, Paper, Scissors using Python's Tkinter library for the graphical user interface. The game allows a player to compete against the computer, with the objective of winning three rounds.

## Files

- **Minigame.py**: Contains the implementation of the game logic, player choices, animations, scoring, and sound effects. The main class is `RPSGame`, which manages the entire game flow.

## How to Run

1. Ensure you have Python installed on your machine.
2. Install the Tkinter library if it is not already included with your Python installation.
3. Download or clone this repository to your local machine.
4. Navigate to the project directory in your terminal.
5. Run the game by executing the following command:

   ```
   python Minigame.py
   ```

## Gameplay Instructions

- Click on one of the buttons labeled "Rock", "Paper", or "Scissors" to make your choice.
- The computer will randomly select its choice.
- The game will display the results of each round, and the first player to win three rounds will be declared the overall winner.
- You can click the "Play Again" button to reset the game after a match.

## Sound Effects

The game includes sound effects for winning, losing, and drawing rounds, providing a more engaging experience.
